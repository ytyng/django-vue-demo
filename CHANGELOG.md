# PyCharm File Watcher
ファイルウォッチャーで自動ビルドが可能。
```
Name: webpack build

File type: Any

Scope: 作る。
statics/src/ 以下を再帰的にウォッチする、という設定が作れる

Program: /usr/local/bin/npm
Arguments: run build
```
これで、Gulp みたいに使える

すでに、index.html は Django から切り離されているので、
Vue CLI でテストサーバ起動してウォッチさせてもいいと思う。
